# homebrew-lib

## brew tap
```bash
brew tap himeyama/lib
```

## brew create
```bash
brew create --tap himeyama/lib <src>.tar.gz
```

## install (Ex.)
    $ brew install himeyama/lib/tcl

## set editor

```bash
# .bashrc
export EDITOR=code
```

### libcuda-convolve
```bash
brew install himeyama/homebrew-lib/libcuda-convolve
```
